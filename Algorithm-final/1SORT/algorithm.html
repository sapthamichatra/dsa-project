QUICK SORTING
Step 1: Define a function quickSort(arr, left, right):

   a. If left is greater than or equal to right, return.

   b. Choose a pivot element from the array, typically the last element (arr[right]).

   c. Call a partition function to rearrange the array elements around the pivot, returning the index of the pivot's final position (pivotIndex).

   d. Recursively call quickSort on the left sub-array: quickSort(arr, left, pivotIndex - 1).

   e. Recursively call quickSort on the right sub-array: quickSort(arr, pivotIndex + 1, right).

Step 2: Define a function partition(arr, left, right):

   a. Choose the pivot element (arr[right]) and set pivotIndex to left.

   b. Iterate from 'i' = left to right-1:

      i. If arr[i] is less than the pivot:

         - Swap arr[i] with arr[pivotIndex].

         - Increment pivotIndex.

   c. Swap arr[pivotIndex] with arr[right] (placing the pivot element in its final sorted position).

   d. Return pivotIndex.

Step 3: Call quickSort(arr, 0, n-1) to sort the entire array, where n is the length of the array.
