Array Representation of Stack( PUSH )
	
Step 1: [Stack already filled?]
	
			IF TOP = MAXSTK ,then 
	
			Print:"Stack overflow" and Return
	
Step 2: Set TOP = TOP + 1
	
Step 3: STACK[TOP]=ITEM
	
Step 4: Return
a.	Input: Number

b.	Process: It stores the number at the beginning of the container and 
checks whether the stack is full.

c.	Output: Enters the number within the container also calculates time 
and space complexity.

Array Representation of Stack( POP )
	
Step 1: [Stack has item to be removed]
	
			IF TOP = MAXSTK then 
	
			Print:"Stack underflow" and Return
	
Step 2: Set ITEM = STACK[TOP]
	
Step 3: Set TOP = TOP - 1
	
Step 4: Return
 
a.	Input: Click on POP Button.

b.	Process: It removes the last entered number and checks whether the stack empty.

c.	Output: Removes the number from the container also calculates time and space complexity.

Linedlist Representation of Stack( PUSH )
	
Step 1: Create a new NODE set NEWNODE = new NODE
	
Step 2: INPUT ITEM
	
Step 3: Set NEWNODE -> DATA = ITEM
	
Step 4: IF TOP = NULL then
	
			a) set TOP = NEWNODE
	
			b) NEWNODE -> LINK = NULL
	
		ELSE
	
			a)set NEWNODE -> LINK = TOP
	
			b)set TOP = NEWNODE
	
		[end of IF ]
	
Step 5: Exit
a.	Input: Number

b.	Process: It creates the node first and allocates memory if the list is empty then the 
node is pushed as the first element of the list otherwise link the previous element to the 
address field of new node.

c.	Output: Pushes the element to the list also calculates time and space complexity.

Linedlist Representation of Stack( POP )
	
Step 1: IF( TOP = NULL) then
	
				Write "STACK IS EMPTY"
	
			ELSE
	
				a)set TEMP = TOP
	
				b)set TOP = TOP -> LINK
	
				c)WRITE: TEMP -> DATA
	
				d)DELETE (TEMP)
	
			[end of IF ]
	
Step 2: Exit
a.	Input: click on pop button.

b.	Process: Node is removed from the end of the linked list. Therefore, must delete 
the value stored in the head pointer. Then the following node will become the head node.

c.	Output: Removes the element to the list also calculates time and space complexity.
